빅데이터 사례 조사



홈쇼핑

1. 목적 : 마케팅

2. 데이터 : 소비자들이 시간대별 검색하는 소비습관, 생활 패턴, 관심분야

3. 분석 내용 : 시간대별 소비자들이 관심을 보이고 소비 욕구가 생기는 상품 분석

4. 가시화 내용 : 저녁식사 시간 전 - 음식 관련 방송, 주말 저녁 - 여행상품 방송, 휴일 전날 - 속옷 방송 등

5. 결론 : 소비자의 생활 습관, 패턴, 관심분야를 파악해 마케팅에 활용하여 매출을 향상 시킬수 있음



병원

1. 목적 : 치료, 질병 예방

2. 데이터 : 환자들의 질병 발생 원인, 증상, 치료법

3. 분석 내용 : 시기별 질병 예측, 증상에 따른 치료법

4. 가시화 내용 : 환자들의 증상을 입력 받아 그에 맞는 치료법을 찾아 주고, 처방 해줌

5. 결론 : 환자들의 질병 및 치료 사례 DATA를 활용하여 즉각적인 치료를 받을 수 있음



대형마트

1. 목적 : 마케팅

2. 데이터 : 소비자의 구매 내역

3. 분석 내용 : 소비자의 관심 분야, 구매 가능 물품

4. 가시화 내용 : 소비자에게 관심분야 팜플렛, 쿠폰 발행등

5. 결론 : 소비자의 구매 내역을 분석하고 팜플렛, 쿠폰등을 발행하여 구매 유도로 매출을 향상 시킴



신규매장

1. 목적 : 동일 업종이 없는 매장 입점

2. 데이터 : 웹상에 나타난 해당 지역에 분포한 매장 종류

3. 분석 내용 : 해당 지역에 있는 동일 업종의 매장 확인

4. 가시화 내용 : 해당 지역에 없는 업종의 매장을 입점하게 되면 타지역으로 유출되는 고객 확보 가능

5. 결론 : 웹상에 나타난 태그를 수집 분석하여 동일 업종이 없는 매장 입점 가능



보험

1. 목적 : 보험가입

2. 데이터 : 가입 고객 정보

3. 분석 내용 : 나이대별, 성별로 가입한 상품 종류

4. 가시화 내용 : 가장 많이 가입한 상품을 그 나이대에 있는 사람들에게 집중 마케팅

5. 결론 : 여러 상품 중 그 사람에게 필요한 상품을 집중적으로 마케팅하여 가입율을 높일 수 있음
